Delete Card Example

Using our nested card example from earlier, we can see how deleting cards can work. By changing the parameter of .deleteCard(deleteChildren), we can see how our nested card changes.

Setting deleteChildren to false will make the immediate subcard child take on the position of the parent child. Here, we see the card's right subcard has become .right.right after deleting .right

JAVASCRIPT
startPoint.right.deleteCard(false)
startPoint.slideRight()
startPoint.right.slideRight()
startPoint.right.slideDown()

Using our nested card example once more, if we do not set deleteChildren, it will automatically pass in as true and delete all subcards of the card called. This can be used to delete the main MultiCard or all children of a subcard.

JAVASCRIPT
startPoint.right.right.deleteCard()
startPoint.slideRight()
startPoint.right.slideRight() //Will cause error as subcard no longer exists
